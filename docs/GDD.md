GAME DESIGN DOCMENT 

CORE CONCEPT
	The core concept of the game is to give players an AI driven historical survival 
simulation experience where the story each player experience is unique to only them. 

MAIN FEATURES
	The story is controlled by an LLM which feeds a machine learning model that estimates
the survival of the player. The whole game is text base with inspiration taken from 
"Be the manager" mobile text based simulation game. Art work is captivating while animations
are left at a minimum. Backgroung sound is kept simple yet relevant (example; my haert will
gone hymn) and all clicking sound intaractions are kept simple to avoid distraction.
	In summary, it is an RPG text based simulation game.
-AI LLM driven story
-Unique story line
-Text base
-RPG
